Highlight examples

This presentation illustrates how to add syntax highlighting.

Use PageDown/PageUp or the arrow keys to move between slides.

A C program

Here is a minimal C program:

#include <stdio.h>
int main() { printf("Hello World!"); }

A Java program

Here is a minimal Java program:

class Hello {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}

A Haskell program

Here is a minimal Haskell program. The Haskell language is not in the default set, so an extra script is included in the head of the presentation. (It appears that the main language name needs to be used, rather than an alias.)

main :: IO ()
main = putStrLn "Hello World!"
